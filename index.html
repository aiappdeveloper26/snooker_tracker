⚙️ One-time setup

Choose a storage backend to sync data between all players' phones.

Free JSON storage. 10,000 lifetime requests — plenty for years of casual play.

  1. Go to jsonbin.io and sign up free
  2. Click CREATE BIN, type exactly: {"friends":[],"sessions":[]} — set to Public
  3. Copy the BIN ID (e.g. 6849ab...)
  4. Paste below and tap Save

Free Postgres database. Unlimited API requests, 500 MB storage — ideal for long-term use.

  1. Go to supabase.com and create a free account + new project
  2. In the Supabase dashboard → SQL Editor, run the SQL setup (see below)
  3. Go to Project Settings → API and copy your Project URL and anon public key
  4. Paste both below and tap Save
