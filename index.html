⚙️ One-time setup

To sync data between all your friends' phones, this app uses a free JSONBin.io bin as shared cloud storage. You only need to do this once.

  1. Go to jsonbin.io and create a free account
  2. Click CREATE BIN, paste in {"friends":[],"sessions":[]}, set visibility to Public
  3. Copy the BIN ID (looks like 6849ab...)
  4. Paste it below and click Save, then commit the HTML to GitHub
